A Django-based web application to manage canteen operations like product listing, ordering, and admin
Features:
-	Product listing
-	Order management
-	Admin panel (Django admin)
-	Media/image upload support
Tech Stack:
-	Python
-	Django
-	SQLite
-	HTML, CSS
Setup:
1.	git clone https://github.com/AnshikaSahu3464/canteen.git
2.	cd canteen
3.	python -m venv venv
4.	venv\Scripts\activate
5.	pip install django
6.	python manage.py migrate
7.	python manage.py runserver
Open: http://127.0.0.1:8000/
